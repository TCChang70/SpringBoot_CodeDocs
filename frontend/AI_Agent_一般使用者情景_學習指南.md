## 什麼是 AI Agent？

**AI Agent（AI 代理人）** 是一種能夠理解你的指令、自動執行多個步驟任務的智慧助理。

與一般 AI 聊天機器人的差異：

| 比較項目 | 一般 AI 聊天 | AI Agent |
|----------|-------------|----------|
| 任務執行 | 只回答問題 | 主動執行多步驟任務 |
| 工具使用 | 無 | 可搜尋網路、操作 App |
| 記憶能力 | 短期對話 | 可追蹤長期目標 |
| 自主程度 | 被動回覆 | 主動規劃並完成 |

---

## 如何下好指令（Prompt）？

有效指令的 **4 個要素**：

```
[角色設定] + [具體任務] + [背景資訊] + [輸出格式]
```

**範例（差 vs 好）：**

| ❌ 模糊指令 | ✅ 清晰指令 |
|------------|------------|
| 「幫我規劃旅遊」 | 「幫我規劃 5 天 4 夜的京都旅遊，預算 3 萬台幣，偏好文化景點，請用表格呈現每日行程」 |
| 「推薦電影」 | 「推薦 3 部類似《寄生上流》的韓國電影，列出導演、年份與一句話簡介」 |
| 「寫信給主管」 | 「幫我寫一封正式請假信給主管，請假原因是家人住院需要陪伴，請假日期 6/5~6/6，語氣需誠懇」 |

## 情景 1：個人行程規劃與時間管理

### 學習目標
學會用 AI Agent 整理行事曆、設定提醒、優化每日時間安排。

### 核心概念
AI Agent 可以：
- 解析你的自然語言描述，轉換為結構化行程
- 偵測時間衝突並主動提醒
- 根據地點、交通建議出發時間

### 實戰練習

**練習 1：整理本週行程**
```
你是我的行程助理。
以下是我本週的事項：
- 週一下午 3 點：線上開會（約 2 小時）
- 週三早上：帶孩子看牙醫（地點：台北車站附近）
- 週五：要交月報給主管（截止中午 12 點前）

請幫我：
1. 整理成行事曆格式（含日期時間）
2. 提醒我需要提早準備的事項
3. 建議週三幾點出發比較合適（假設住在板橋）
```

**練習 2：每週計畫優化**
```
我每週一早上需要規劃這週工作。
我的工作時間是早上 9 點到下午 6 點。
我有以下習慣：午休需要至少 45 分鐘，下午 3 點後創意工作效率較差。

請根據這些條件，幫我設計一個「最佳每週時間分配範本」。
```

### 常見工具
- **Apple Siri** + 日曆 App
- **Google Assistant** + Google 日曆
- **ChatGPT** 自訂 GPT + 行事曆外掛

### 學習重點
> 提供越具體的時間、地點與習慣資訊，AI 的安排越貼近你的需求。

## 情景 2：購物比價與採購建議

### 學習目標
學會利用 AI Agent 篩選商品、比較規格，避免衝動購物。

### 核心概念
AI Agent 在購物場景的價值：
- 根據預算與需求過濾選項
- 客觀比較規格（不受廣告影響）
- 分析評價找出真實使用者回饋

### 實戰練習

**練習 1：耳機採購決策**
```
我想買一副藍牙耳機，條件如下：
- 預算：5000 台幣以下
- 主要用途：通勤聽音樂 + 視訊會議
- 必備功能：主動降噪（ANC）
- 偏好：重量輕、續航 20 小時以上

請推薦 3 個選項，用表格比較品牌、售價、續航、降噪效果、重量，
最後給我一個你的推薦理由（100字以內）。
```

**練習 2：防衝動購物清單**
```
我想買以下東西，請幫我分析哪些是「真正需要」還是「衝動購物」：
- 空氣炸鍋（家裡已有烤箱）
- 新的運動鞋（舊的快破了）
- 第三個行動電源
- 升級 4K 螢幕（目前 1080p 用來辦公）

請逐項分析，並給出建議購買時機。
```

### 學習重點
> 清楚說明「用途」比說「我要最好的」更能得到實用建議。

## 情景 3：旅遊規劃與訂票

### 學習目標
從零開始，讓 AI 幫你規劃完整旅程，包含交通、住宿、景點、餐廳。

### 核心概念
完整旅遊規劃需要提供：
1. **目的地**與**天數**
2. **預算範圍**
3. **旅遊風格**（文化/美食/自然/購物）
4. **同行人數與年齡**
5. **特殊需求**（無障礙/素食/攜帶寵物）

### 實戰練習

**練習 1：京都 5 天行程**
```
幫我規劃一趟京都旅遊，條件如下：
- 天數：5 天 4 夜
- 出發地：台北（飛關西機場）
- 旅伴：夫妻兩人，無小孩
- 預算：總共 3 萬台幣（含機票）
- 偏好：傳統文化、寺廟神社、抹茶甜點
- 避免：過多購物景點

請提供：
1. 每日行程表（含景點、午晚餐建議、交通方式）
2. 住宿區域建議
3. 預算分配表
4. 3 個省錢小技巧
```

**練習 2：國內一日遊**
```
幫我規劃台南一日遊，從台北出發，搭高鐵。
我喜歡：台灣小吃、老街歷史、下午茶咖啡廳。
回程希望在傍晚 6 點前搭車。
請規劃含時間點的詳細行程。
```

### 學習重點
> 先問 AI「我適合去哪裡？」再問「怎麼去？」，能獲得更個人化建議。

## 情景 4：文件撰寫與文字處理

### 學習目標
用 AI 提升各種文書工作品質，節省撰寫時間。

### 核心概念
文件寫作常見任務分類：

| 類型 | 範例 | 關鍵提示 |
|------|------|----------|
| 正式信件 | 請假信、投訴信 | 說明收件人身分與關係 |
| 履歷文件 | 自傳、求職信 | 提供個人背景與目標職位 |
| 摘要整理 | 會議記錄、讀書心得 | 貼上原文，指定摘要長度 |
| 翻譯潤稿 | 中翻英、改善語氣 | 指定語氣風格（正式/友善） |

### 實戰練習

**練習 1：請假信**
```
請幫我寫一封請假信，內容如下：
- 收件人：直屬主管（關係：一般職場，非親近）
- 請假原因：父親突然住院需要陪伴照顧
- 請假日期：6月5日（星期四）至6月6日（星期五），共 2 天
- 工作交接：重要任務已交代同事 A 代理
- 語氣要求：誠懇、不誇張、正式但不冷漠

請用繁體中文撰寫，200 字以內。
```

**練習 2：會議記錄摘要**
```
以下是會議的逐字稿片段，請幫我整理成正式會議記錄：
[貼入逐字稿]

輸出格式：
- 會議主題：
- 出席人員：
- 討論重點（條列式）：
- 決議事項：
- 待辦事項（含負責人與期限）：
```

**練習 3：潤飾語氣**
```
請將以下的文字改寫得更專業、正式，適合寄給客戶：
「那個問題昨天已經修好了，你們可以再試試看，應該沒問題了。」
保留原意，繁體中文，語氣要自信但有禮貌。
```

### 學習重點
> 告訴 AI「收件對象是誰」以及「你們的關係」，語氣才會恰當。

## 情景 5：健康與醫療諮詢輔助

### 學習目標
善用 AI 獲取健康資訊、管理用藥提醒，但了解其限制。

### 核心概念

> ⚠️ **重要聲明：AI 不能取代醫師！**  
> AI 提供的是「參考資訊」，不是「診斷」。出現症狀請務必就醫。

AI Agent 在健康領域**可以做的事**：
- 解釋醫療名詞（檢查報告看不懂時）
- 建議應該掛哪一科
- 設定用藥提醒
- 整理健康數據趨勢

AI Agent **不應該做的事**：
- 自行診斷疾病
- 建議自行調整藥量
- 替代急症的緊急處置

### 實戰練習

**練習 1：症狀初步查詢**
```
我最近兩週有以下症狀：
- 早上起床頭痛（後腦勺）
- 偶爾頭暈
- 睡眠品質變差

我沒有高血壓，不抽菸，每天坐辦公室工作 8 小時。

請問：
1. 可能的原因有哪些（列出 3~5 種）
2. 建議掛哪一科
3. 就醫前可以先做哪些自我觀察記錄
```

**練習 2：健康檢查報告解讀**
```
請幫我解釋以下健康檢查數值的意義，並告訴我哪些需要注意：
- 空腹血糖：105 mg/dL
- 總膽固醇：215 mg/dL
- HDL 膽固醇：45 mg/dL
- LDL 膽固醇：148 mg/dL
- BMI：26.3

請用一般人能理解的語言說明，並給出生活改善建議。
```

**練習 3：設計用藥提醒計畫**
```
我需要每天服用以下藥物，幫我設計提醒計畫：
- 降血壓藥：每天早上飯後 1 顆
- 維生素 D：每天任意時間 1 顆
- 魚油：每天晚上飯後 2 顆

我的早餐時間約 7:30，晚餐時間約 7:00。
請設計一張每日用藥時間表，並說明各藥物需要注意的事項。
```

### 學習重點
> 越詳細描述自己的生活習慣與病史，AI 給的建議越有參考價值。

## 情景 6：學習輔助與家教

### 學習目標
讓 AI 成為你或孩子的 24 小時個人家教。

### 核心概念
AI 家教的最大優勢：**無限耐心、隨時可問、依程度調整說明深度**。

有效學習互動的技巧：
1. 先說明**目前的程度**
2. 要求用**比喻或例子**解釋
3. 請 AI **出題**來測試自己
4. 答錯時要求**詳細解說**，不是只給答案

### 實戰練習

**練習 1：概念學習（國小/國中程度）**
```
我是小學 5 年級學生，請用我能理解的方式解釋「光合作用」：
1. 用一個生活中的比喻解釋
2. 告訴我植物需要什麼、產生什麼
3. 用圖示（文字版）表示過程
4. 最後出 2 題選擇題考考我
```

**練習 2：英文練習（對話模式）**
```
請扮演我的英文對話練習夥伴。
我的程度：國中英文，基本日常對話。
今天主題：在餐廳點餐。

規則：
- 你先用英文說一句話（餐廳服務生角色）
- 我用英文回覆
- 如果我有語法錯誤，在我回覆後用括號標示並說明
- 對話進行 5 輪
```

**練習 3：考前衝刺**
```
我明天要考「台灣史 清朝統治時期」這個單元。
請幫我：
1. 整理最重要的 10 個考點（條列式）
2. 出 5 題選擇題（附答案與解說）
3. 出 2 題簡答題（附參考答案）
4. 告訴我最容易混淆的觀念是什麼
```

### 常見工具
- **Khan Academy / Khanmigo** - 數學、科學線上家教
- **Duolingo** - 語言學習
- **ChatGPT / Claude** - 任何學科的客製化學習

### 學習重點
> 「請出題考我」比「幫我解釋」更能有效驗證學習成果。

## 情景 7：家庭財務管理

### 學習目標
用 AI 建立記帳習慣、分析消費模式、制定儲蓄目標。

### 核心概念
AI 財務助理的三個層次：

```
Layer 1：記錄（記帳）
    ↓
Layer 2：分析（看懂你的消費習慣）
    ↓
Layer 3：建議（如何存更多錢）
```

### 實戰練習

**練習 1：月度消費分析**
```
以下是我 5 月份的消費記錄，請幫我分析：

餐飲：咖啡廳 850元、超商 1200元、外送 3600元、餐廳聚餐 2400元
交通：捷運 890元、Uber 1350元
購物：衣服 3200元、日用品 680元
娛樂：Netflix 390元、KTV 800元
醫療：藥局 450元

我的月收入：38,000 元
固定支出（未列入）：房租 12,000元、電話費 699元

請：
1. 幫我分類統計各項目金額與占比
2. 指出消費異常或偏高的項目
3. 建議 3 個可以立即節省的方法
4. 計算這個月的儲蓄率
```

**練習 2：儲蓄目標規劃**
```
我的財務狀況：
- 月薪：38,000 元
- 每月固定支出：約 22,000 元
- 目標：3 年後存夠 30 萬元買車頭期款

請幫我：
1. 計算目前每月需存多少錢
2. 評估這個目標是否可行
3. 如果無法達成，需要如何調整（增加收入或減少支出）
4. 設計一個「存錢挑戰計畫」
```

### 學習重點
> 定期（每週或每月）把消費記錄貼給 AI 分析，比一次性規劃更有效。

## 情景 8：娛樂內容推薦

### 學習目標
讓 AI 成為你的個人娛樂策展人，找到真正符合你口味的內容。

### 核心概念
提升推薦品質的關鍵：
- 說明**喜歡某作品的哪個部分**（不只說「我喜歡 A」）
- 說明**不喜歡什麼**（負面條件同樣重要）
- 提供**目前心情或需求**

### 實戰練習

**練習 1：電影推薦**
```
我喜歡《寄生上流》，因為我喜歡：
- 社會議題探討
- 出乎意料的劇情轉折
- 黑色幽默的呈現方式

我不喜歡：純粹暴力血腥、過於正能量的結局

請推薦 5 部類似的電影（不限國籍），
每部附上：導演、年份、一句話介紹、適合什麼心情下觀看。
```

**練習 2：根據心情找音樂**
```
今天我工作壓力很大，想在下班後放鬆但又不想太嗜睡。
我平常喜歡的音樂：獨立流行、日系音樂、偶爾聽爵士。
不喜歡：太激烈的電子音樂、鄉村音樂。

請推薦一個適合今晚的播放清單主題，
以及 8 首具體歌曲（含歌手名）。
```

**練習 3：書單推薦**
```
我是 30 歲上班族，每天通勤 40 分鐘，想利用這段時間聽 Podcast 或讀書。
興趣：心理學、時間管理、個人成長、偶爾喜歡懸疑小說。
最近讀過且喜歡的：《原子習慣》、《被討厭的勇氣》。

請推薦 5 本書與 3 個 Podcast，
說明為何適合我以及大約需要多少時間讀完/聽完。
```

### 學習重點
> 告訴 AI 你「喜歡的原因」比告訴它「喜歡的作品名稱」更能得到精準推薦。

## 情景 9：客服與問題解決

### 學習目標
用 AI 協助處理客訴、查詢服務狀態、撰寫申訴信件。

### 核心概念
AI 在客服場景的角色：
- **查詢整理**：幫你整理問題，讓真人客服更快理解
- **文字代勞**：幫你寫投訴信、申訴書
- **情緒緩衝**：把憤怒的情緒轉化為理性表達

### 實戰練習

**練習 1：撰寫投訴信**
```
請幫我寫一封正式的客訴信，情況如下：
- 公司：某外送平台
- 問題：訂單顯示已送達，但我根本沒收到，金額 350 元
- 聯絡過客服：有，客服說「會調查」但已過 3 天沒有回音
- 我的訴求：全額退款，並希望對方說明原因

語氣要求：堅定、理性、有禮但明確表達不滿。
格式：正式書信，繁體中文。
```

**練習 2：退貨申請準備**
```
我想退一台購買 20 天的電風扇，原因是：
- 商品描述說「靜音設計」但實際噪音很大
- 這屬於「商品與描述不符」

請幫我：
1. 整理退貨的合法依據（消費者保護法）
2. 準備好的溝通話術（給客服說的）
3. 如果客服拒絕，下一步可以找哪些單位申訴
```

### 學習重點
> 與真人客服溝通前，先讓 AI 幫你「整理訴求」，可大幅提升溝通效率。

## 情景 10：社群媒體與內容創作輔助

### 學習目標
用 AI 提升個人社群貼文質量，打造獨特的個人風格。

### 核心概念
AI 社群創作的工作流程：

```
提供素材（照片描述/主題/心情）
    ↓
AI 生成初稿
    ↓
你加入個人風格修改
    ↓
發布
```

### 實戰練習

## 綜合練習：AI Agent 使用自我評估

完成以下 10 道題目，檢驗你對 AI Agent 的掌握程度：

1. **行程規劃**：告訴 AI 你明天的行程，請它幫你判斷時間是否有衝突
2. **購物決策**：描述一個你最近想買的東西，請 AI 幫你分析是否值得買
3. **旅遊諮詢**：問 AI 你下個假期（3 天以內）適合去哪裡，提供你的喜好
4. **文件撰寫**：請 AI 幫你寫一封感謝信（對象自選）
5. **健康查詢**：問 AI 最近你有的一個輕微症狀，請它建議應掛什麼科
6. **學習問題**：問一個你最近不太懂的概念，並要求它舉例說明
7. **財務規劃**：告訴 AI 你這個月的餐飲花費，問它是否偏高
8. **娛樂推薦**：說出你喜歡的一部作品，請它推薦 3 個類似的
9. **客服協助**：描述一個近期消費不滿意的經驗，請 AI 幫你寫回饋意見
10. **內容創作**：用一句話描述你最近的生活，請 AI 幫你寫成 IG 貼文

## 使用 AI Agent 的重要原則

### ✅ 正確使用方式
- 把 AI 視為**工具**，不是全知的神
- 對重要決策（醫療、法律、財務）**務必諮詢專業人士確認**
- 敏感個資（身份證、密碼、帳號）**絕對不要輸入**
- 養成**查核 AI 輸出**的習慣，特別是數字與引用

### ❌ 常見錯誤
- 指令太模糊，期待 AI 猜測你的需求
- 完全相信 AI 的回答，不加以驗證
- 把個人隱私資料輸入 AI 系統
- 只用一次，沒有持續優化提示方式

### 隱私安全提醒

| 可以輸入的資訊 | 不應輸入的資訊 |
|--------------|--------------|
| 一般性問題與需求 | 身份證字號 |
| 匿名化的消費記錄 | 銀行帳號密碼 |
| 公開的個人興趣 | 醫療病歷詳細資料 |
| 工作一般性問題 | 公司機密文件 |
