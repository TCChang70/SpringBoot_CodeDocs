## 內容優化建議

### 1. 關聯映射類型部分
- **增加關聯關係圖**：用 ER 圖展示不同關聯類型
- **增加實際應用場景**：每個關聯類型的典型使用場景
- **增加關聯映射的最佳實踐**：什麼時候使用哪種關聯

### 2. 一對多/多對一實作部分
- **增加完整的實作範例**：從 Entity 到 Controller 的完整實作
- **增加關聯操作的注意事項**：如雙向關聯的維護
- **增加關聯資料的 CRUD 操作**：如何正確操作關聯資料

### 3. FetchType 策略部分
- **增加 LAZY 和 EAGER 的詳細比較**：效能影響和使用場景
- **增加 Hibernate Proxy 的說明**：理解 LAZY 載入的原理
- **增加 FetchType 的選擇指南**：如何選擇合適的 FetchType

### 4. N+1 問題部分
- **增加 N+1 問題的詳細解釋**：為什麼會產生 N+1 問題
- **增加 JOIN FETCH 的完整範例**：多種 JOIN FETCH 的使用方式
- **增加 @EntityGraph 的進階用法**：複雜的 EntityGraph 配置
- **增加其他解決方案**：如 @BatchSize、Subselect 等

### 5. @Query 進階部分
- **增加更多複雜查詢範例**：如子查詢、聚合查詢、多表查詢
- **增加查詢效能優化**：如何優化複雜查詢
- **增加原生 SQL 查詢**：何時使用原生 SQL

### 6. 級聯操作部分
- **增加級聯操作的詳細說明**：每種 CascadeType 的作用
- **增加級聯操作的風險**：不當使用可能導致的問題
- **增加級聯操作的最佳實踐**：如何安全使用級聯操作

## 實作練習文件結構

建議創建一個獨立的實作練習文件，包含：

1. **基礎練習**：關聯映射基本實作
2. **進階練習**：N+1 問題解法和複雜查詢
3. **實戰練習**：完整的關聯資料管理
4. **除錯練習**：解決常見問題

這些優化建議可以幫助學習者更全面地理解 JPA 關聯映射的知識，並提供足夠的實作機會來鞏固學習成果。
